Distance Map Generator

You can open the tool via "Window / Catlike Coding / Distance Map Generator".

You can access the online documentation via "Help / Catlike Coding / Distance Map Generator".

If you want to access CCTextureMapGenerator in Javascript, put the "Distance Map Generator" folder into the "Plugins" folder.
If there's no "Plugins" folder in your project, simply create one.

Keep the licence in mind if you consider including this tool in a distribution. Make sure you fully understand the GNU GPL.
